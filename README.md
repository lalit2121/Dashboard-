---

## 📊 Project Overview
This project showcases an interactive **Airline KPI Dashboard** built using **Power BI**, based on raw airline operational data.  

The dashboard transforms raw data into meaningful insights to monitor performance, efficiency, and key airline operations.

---

## 🎯 Objectives
- Convert raw airline data into actionable insights  
- Track key airline KPIs in a single dashboard  
- Improve visibility into operational performance  
- Enable quick decision-making through visual analytics  

---

## 📌 Key Features
- ✈️ Flight Performance Tracking (delays, cancellations, on-time performance)  
- 📉 Delay Analysis (departure & arrival delays)  
- 📊 Passenger & Load Metrics  
- ⏱️ Operational Efficiency KPIs  
- 🌍 Route & Airport Insights  
- 📈 Interactive filters and slicers  

---

## 🛠️ Tools & Technologies
- Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Excel / CSV (Raw Data Source)  

## 🔍 Key Insights
- Identification of high-delay routes  
- Trends in on-time performance  
- Peak operational periods  
- Route and airport performance comparison  
- Operational bottlenecks  

---

## 🚀 How to Use
1. Open the `.pbix` file in **Power BI Desktop**  
2. Load or refresh the dataset  
3. Explore the dashboard using filters and slicers  
4. Analyze KPIs and extract insights
